## 📖 Overview

**AI-Educator** is a modern web application designed to help students, developers, creators, professionals, and AI enthusiasts discover and learn the best AI tools available today.

Instead of simply listing AI websites, AI-Educator provides a structured learning experience by combining curated AI tool recommendations, official links, usage guides, tutorial videos, intelligent search, and voice-enabled accessibility.

The platform makes learning AI simple by organizing tools into specialized categories and providing detailed information for every tool.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/umar-patel07/AI-Educator.git
```

Navigate into the project

```bash
cd AI-Educator
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
